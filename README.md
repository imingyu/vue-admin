# VueAdmin —— 一个前端的灵活高效的后台管理系统开发集成库

## 安装
``` bash
git clone https://github.com/imingyu/vue-admin.git
cd vue-admin
npm i
```

## 运行
``` bash
# 开启本地服务器，访问：http://localhost:8090
npm run dev
```

## 编译
``` bash
npm run build
```

## 测试
``` bash
# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
